# Credit Risk Prediction Model

This project predicts whether a customer will default on their credit card payment.

## Features used:
- Credit Limit (LIMIT_BAL)
- Credit Utilization
- Average Payment Delay

## Model:
- Logistic Regression (optimized for recall)

## Deployment:
- Streamlit app for real-time prediction

## Key Insight:
- High recall prioritized to minimize missed defaulters
